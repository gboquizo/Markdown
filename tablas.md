### Tablas

Las tablas se crean de una manera tan sencilla como delimitando las columnas con el caracter pipe o tuberia ( | ) y guiones ( - ) para las filas. Ej.:

      |Encabezado1|encabezado2|
      |-----------|-----------|
      |contenido1 |contenido2 |

|Encabezado1|encabezado2|
|-----------|-----------|
|contenido1 |contenido2 |
